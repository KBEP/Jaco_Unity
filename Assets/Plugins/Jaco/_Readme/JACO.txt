Система JACO помогает делать корректый перевод динамически изменяемого текста. Вставлять в текст слова в правильной форме в зависимости от рода, падежа, числа и др. модификаторов.



Interpreter
Парсит строку, выполняя замены текста согласно языку ~x~{y|z}, см. Текущий вариант синтаксиса.txt.

XSV
Обеспечивает чтение данных формата TSV/CSV из файла, строки или потока, поддерживает формат Google.Docs, OpenOffice, LibreOffice. Форматы др. редакторов не тестировались.

MultiArray
Структура данных для хранения значений, доступных по нескольким ключам, своего рода N-мерный словарь фиксированного размера.

WordGeneration
Генерирует слова на основе шаблона (+~-синтаксис), см. Word Form Formats.txt.

WordFormGeneration
Генерирует структуру данных для хранения словоформ, читая значения из файла, строки или потока.
